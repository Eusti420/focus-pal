# FocusPal 🦊

Dein persönlicher ADHS-Begleiter.

## Features
- ✅ Aufgaben mit Tages- und Wochenansicht
- ⚡ Quick Wins mit XP-Belohnungssystem
- 🏆 Level-System (50 Level + Prestige)
- ⏱️ Fokus-Timer (Pomodoro)
- 🔔 Erinnerungen & Benachrichtigungen
- 🧠 ADHS-Kompensationsstrategien
- 📚 Wissen über ADHS, Ernährung, Supplemente
- 📱 Installierbar als PWA (iOS & Android)

## Technologie
- Vanilla HTML / CSS / JavaScript
- PWA mit Service Worker
- localStorage (keine externe Datenbank)

## Installation
Safari → Teilen → „Zum Home-Bildschirm"
